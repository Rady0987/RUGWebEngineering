get /actors
Fetch all actors in the dataset, optionally filtered by name. (actorsGet)

Query parameters

actorName (optional)
Query Parameter — Actor name to filter by

Return type

array[Actor]

Example data

Content-Type: application/json
[ {
  "name" : "name"
}, {
  "name" : "name"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • text/csv
  • application/json

Responses

200

successful operation

post /movies
Add a movie to the dataset by it's unique URL. (moviesPost)

Query parameters

url (required)
Query Parameter — The unique IMDB url of a movie
title (optional)
Query Parameter — The title of the movie
rating (optional)
Query Parameter — The rating of the movie
year (optional)
Query Parameter — The release year of the movie
users_rating (optional)
Query Parameter — The user rating for the movie
votes (optional)
Query Parameter — The amount of votes for the movie
metascore (optional)
Query Parameter — The metascore of the movie
img_url (optional)
Query Parameter — The image URL of the movie
countries (optional)
Query Parameter — The countries of first release
languages (optional)
Query Parameter — The languages in which the movie is available
actors (optional)
Query Parameter — The actors in the movie
genre (optional)
Query Parameter — The genres of the movie
tagline (optional)
Query Parameter — The tagline of the movie
description (optional)
Query Parameter — The description of the movie
directors (optional)
Query Parameter — The directors of the movie
runtime (optional)
Query Parameter — The runtime of the movie

Responses

200

successful operation

405

invalid arguments
